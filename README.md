# 🧠 MedClassify

## 📌 Overview
MedClassify is an AI-powered project for medical question type classification. It categorizes healthcare-related questions into four classes:

- 🔬 Diagnostic  
- 📚 Factoid  
- 🛡️ Preventive  
- 📈 Prognostic  

This project demonstrates the use of Natural Language Processing (NLP) and transformer-based models for understanding medical queries.

---

## 🤖 Models

- **Baseline Model:** BERT  
- **Advanced Model:** MedGemma (LoRA fine-tuning)

---

## 📊 Dataset

- Training Samples: 56,142  
- Test Samples: 2,474  
- Domain: Mental Health (Anxiety, Depression, OCD, etc.)

---

## ⚙️ Features

- Data preprocessing and cleaning  
- Data quality improvement  
- Class imbalance handling  
- Transformer-based training  
- Model evaluation (accuracy, classification report)  
- Confusion matrix visualization  
- Sample predictions  

---

## 📈 Results

- BERT Accuracy: ~79.8%  
- MedGemma: Improved domain understanding  
- Better performance on Diagnostic & Preventive classes
